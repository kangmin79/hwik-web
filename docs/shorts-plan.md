# 유튜브 숏츠/릴스 자동 생성 계획

## 콘텐츠 방향
- AI 생성 시네마틱 드론 뷰 (매번 다른 영상)
- 데이터 카드 오버레이 (danji_pages 기반)
- 감성적 BGM
- 15초 완성

## 콘텐츠 유형
1. 랭킹형: "서울 평당가 TOP 5", "전세가율 위험 단지"
2. 비교형: "강남 vs 서초", "잠실엘스 vs 래미안"
3. 퀴즈형: "이 아파트 얼마일까요?"
4. 시세 속보형: "이번주 최고가 거래"

## 영상 구성 (15초)
- 0~3초: 서울 상공 드론 뷰 (AI 생성)
- 3~7초: 해당 구/동으로 줌인 (AI 생성)
- 7~12초: 데이터 카드 오버레이 (단지명/가격/전세가율)
- 12~15초: 휙 로고 + "실시간 시세는 휙에서"

## 기술 스택
- AI 영상: Sora / Runway / Kling (월 $12~40)
- 데이터: danji_pages DB (5,856개 단지)
- 카드 생성: Python PIL (1080×1920 세로)
- 영상 합성: moviepy (이미지+영상+BGM)
- 자막: 자동 생성

## 자동화 목표
- 매일 1개 자동 생성 (5,856개 = 16년치)
- 단지 좌표 → AI 프롬프트 자동 생성
- 데이터 카드 자동 렌더링
- BGM 랜덤 선택

## 실행 시기
- MVP 완성 후
- 유튜브/인스타 채널 개설 후

## 참고
- 같은 배경 재활용 금지 (매번 다른 AI 영상)
- AI 영상 퀄리티가 핵심 (음악 채널 수준)
