## 1. 한 줄 요약

> **"광고비 0원, AI가 매칭해주는 부동산 플랫폼"**

중개사가 매물 정보를 말하듯이 적으면, AI가 아름다운 카드로 만들어주고,
손님에게 카카오톡으로 "휙" 보내는 에이전틱 SaaS.

---

## 2. 풀고자 하는 문제

### 중개사의 현실

| 문제 | 현재 방식 | 비용 |
|------|-----------|------|
| 매물 노출 | 직방/다방/네이버 광고비 순위 | 월 50~200만원 |
| 매물 등록 | 사진 10장 + 정보 30개 항목 수동 입력 | 매물당 20~30분 |
| 손님 매칭 | 전화/문자/단톡방 수동 공유 | 하루 2~3시간 |
| 시세 파악 | 국토부 사이트 직접 검색 | 매번 10분+ |
| 경쟁 | 광고비 더 많이 쓴 중개사가 상위 노출 | 영세 중개사 불리 |

### 핵심 인사이트

- 한국 공인중개사 **75,000명+** (서울+인천+경기만 32,000명+)
- 중개사 월 광고비 평균 **90~200만원** (직방/다방/네이버)
- 광고비 = 매물 품질이 아닌 **지불 능력** 순위
- 허위매물 비율 추정 **30~40%** (광고 인센티브 구조적 문제)
- 손님은 "목록 → 상세 → 뒤로가기" 반복에 지침

## 3. 솔루션: 휙 (Hwik)

### 핵심 가치

```
기존: 중개사가 30분 걸려 매물 등록 → 광고비로 노출 경쟁
휙:   중개사가 30초 텍스트 입력 → AI가 카드 생성 → 손님에게 카톡으로 "휙"
```

### 제품 구조

```
┌─────────────────────────────────────────────┐
│                  휙 플랫폼                     │
│                                              │
│  [중개사 허브]          [손님 카드뷰어]         │
│   ├─ AI 매물 등록       ├─ 스와이프 비교        │
```

### 4-3. 매칭 알고리즘 (125점 만점)

```
거래유형 일치:  40점 (불일치 = 즉시 탈락)
가격 범위:     30점 (5% 초과당 -2점)
위치 근접도:   20점 (같은 동=20, 1km=15, 3km=10)
매물 유형:     15점 (아파트/오피스텔/원룸 등)
태그 매칭:     20점 (필수=10, 선호=5)
제외 태그:    -50점 (즉시 탈락)
임베딩 보너스:  +5점 (태그 약할 때 의미 유사도)
─────────────────
매칭 기준:     80점+ = 강한 매칭
```

## 7. 경쟁 분석

### 7-1. 기존 플랫폼과 비교

| 항목 | 직방 | 다방 | 네이버 | **휙** |
|------|------|------|--------|--------|
| **중개사 월 광고비** | 90~200만 | 60~100만 | 100만+ | **0원** |
| **매물 노출 기준** | 광고비 순위 | 광고비 순위 | 광고비 순위 | **AI 매칭** |
| **매물 등록 시간** | 20~30분 | 20~30분 | 30분+ | **30초** |
| **데이터 출처** | 중개사 자가 입력 | 중개사 자가 입력 | 중개사 자가 입력 | **국토부 공식 데이터** |
| **허위매물 구조** | 광고 인센티브 有 | 광고 인센티브 有 | 광고 인센티브 有 | **인센티브 無** |
| **손님 경험** | 목록→상세→뒤로 반복 | 목록→상세→뒤로 반복 | 목록→상세→뒤로 반복 | **스와이프 비교** |
| **AI 기능** | 없음 | 없음 | 일부 추천 | **파싱+매칭+검색** |
| **중개사 CRM** | 없음 | 없음 | 없음 | **손님 관리+매칭** |

### 7-2. 경쟁 우위 (Moat)

1. **데이터 해자**: 13,418 단지 × 36개월 실거래 = 경쟁사 1년+ 걸리는 수집량
2. **AI 파이프라인**: 텍스트→카드→매칭→검색 전체 자동화 (경쟁사 없음)
3. **네트워크 효과**: 중개사↔손님↔데이터 삼각 루프
4. **비용 구조**: 월 10만원 인프라로 만 명 서비스 가능 (서버리스)
5. **SEO 자산**: 11,841 페이지가 매일 자동 갱신 (경쟁사 따라오기 어려움)

### 7-3. 왜 기존 플랫폼이 이걸 안 하는가?

```
직방/다방의 수익 = 광고비 순위 매출
    ↓
AI 매칭을 도입하면 = 광고비 모델이 무너짐
    ↓
기존 플랫폼은 구조적으로 이 혁신을 할 수 없음
= 클래식 "이노베이터의 딜레마"
```

## 16. 왜 지금인가?

1. **AI 기술 성숙**: Claude/GPT가 매물 텍스트를 정확히 파싱할 수 있는 수준 도달
2. **정부 데이터 개방**: 국토부 실거래가 API 안정화 → 신뢰할 수 있는 데이터 소스
3. **에이전틱 SaaS 트렌드**: AI가 "도구"가 아니라 "에이전트"로 작동하는 시대
4. **부동산 광고비 불만 최고조**: 중개사 커뮤니티에서 광고비 부담 호소 급증
5. **모바일 카드 UX 보편화**: 틴더/당근마켓 등으로 스와이프 UX에 익숙한 사용자층
