### 1. DB: danji_pages 테이블
- `sql/danji_pages.sql` — DDL + 샘플 데이터

| 컬럼 | 용도 |

| recent_trade | 평형별 최근 실거래가 |
| all_time_high | 평형별 역대 최고가 |

| active_listings | 휙 등록 매물 |
| seo_text | 하단 SEO 텍스트 |
