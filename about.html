휙이 하는 일

휙은 공인중개사가 손님에게 매물을 빠르고 정확하게 보낼 수 있도록 돕는 부동산 도구입니다.

매물 원고를 말하듯 적으면 AI가 자동으로 파싱하고, 손님 조건에 맞는 매물을 자동으로 매칭합니다. 카드 한 장에 매물의 모든 정보가 담겨 있어, 손님은 스와이프만으로 매물을 비교할 수 있습니다.

국토교통부 실거래가 공개시스템의 공식 데이터를 기반으로 합니다. 매일 새벽 자동으로 최신 거래를 수집하고 집계합니다. 추측이나 계산 데이터는 사용하지 않습니다.

주요 기능

  • 단지 시세 페이지 — 서울 6,000개 단지의 매매/전세/월세 실거래가, 시세 차트, 주변 시설 정보
  • 매물 카드 — 한 카드에 매물의 모든 핵심 정보. 스와이프로 빠른 비교
  • AI 매칭 — 손님 조건에 맞는 매물을 자동으로 찾아 추천
  • 매물 검색 — 자연어로 검색하면 SQL 기반 정확 매칭
  • 손님 관리 — CRM, 일정, 알림 기능으로 체계적 관리
  • 블로그 원고 — 단지 데이터 기반 블로그 원고 자동 생성

데이터 출처

  • 실거래가 — 국토교통부 실거래가 공개시스템
  • 공급면적 — 국토교통부 건축물대장 API
  • 단지 정보 — K-apt 공동주택 관리정보시스템
  • 지하철/학교 — 공공데이터포털
